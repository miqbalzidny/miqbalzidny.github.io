Detection is carried out by processing image which is then the result of image processing is classified. Automatic Plant Watering Machine

During this time, water watering is done manually. However, sometimes we don't have enough time to water plants. Therefore, we need a tool that can help relieve the activity of watering plants. The tool is a system that can work automatically. This tool uses using two water pumps as a water supplier to be watered and DC motorbikes as a driver of the ship. The target plants of this tool are crops such as chili, eggplants, cucumbers, long beans, and others.

Smart Pispot as Monitoring Volume and Demanding Urine Based on Internet of Things

Smart Pispot is a robot prototype that can deliver a pispot that contains urine, measures the volume of urine, and throws it into a landfill integrated with the Internet of Things. To remove urine automatically into the bathroom a path is made like a line follower to the bathroom and after the robot reaches the pispot bathroom will be removed by the robot's hand and the urine is discharged into the drain.
